# Librarian

A GoAdmin plugin which see markdown docs.

![](http://quick.go-admin.cn/docs/librarian_interface_2.png)

## How To

An example: 

```
cd example
GO111MODULE=on go run main.go
```

visit: [http://localhost:9033/admin/librarian](http://localhost:9033/admin/librarian)
